# Starplux

Starplux is a multi-tenant business operations platform built with Laravel 11 and Livewire 3.

It helps teams manage day-to-day operations from a single dashboard, including:

- Invoicing and customer tracking
- Analytics and reporting
- Jobs and task management
- Departments and user administration
- Product and bookkeeping workflows
- Role-based dashboards (tenant admin, department admin, user)
- Master admin tenant oversight

This repository includes latest releases for Starplux desktop application for Windows and Linux users.
